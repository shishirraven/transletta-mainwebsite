Location Information<\/h2>\n\n\n\n

Some of the services may use location information transmitted from users’ mobile phones. We only use this information within the scope necessary for the designated service. We use location information from the users to find and assign jobs for the customers and shovelers which is nearby to them. Location is revealed only to the service provider & to the customer.<\/p>\n\n\n\n
